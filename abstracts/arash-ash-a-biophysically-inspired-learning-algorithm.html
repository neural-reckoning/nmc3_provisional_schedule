A Biophysically Inspired Learning Algorithm for Deep Neural Networks

Arash Ash

Abstract

Backpropagation (BP) of global error has been the only way to learn artificial deep neural networks efficiently. However, BP in the traditional form is not biologically plausible since the feedback path requires symmetric weights of feedforward. Inspired by the Hebbian plasticity idea of learning synaptic weights from neuronal activities, we propose a novel learning algorithm called Feedback Propagation (FP) that uses a feedback network learned by the changes in the local activations. We show that this results in a local negative feedback loop that produces credit assignments in the global gradient direction. We trained deep, fully connected, and convolutional models on MNIST and CIFAR image recognition datasets with BP and FP algorithms and compared the performances and the learned representations. The results show that FP achieves similar generalization error as BP on fully connected models. However, the convolutional models did not learn as well using the FP algorithm.
