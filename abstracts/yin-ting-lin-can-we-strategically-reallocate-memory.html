Can we strategically reallocate memory resources?

Yin-ting Lin

Daryl Fougnie, New York University Abu Dhabi

Abstract

We can selectively prioritize information during perception. In working memory, information can be similarly prioritized by a retro-cue. This is thought to occur due to reallocation of resources to prioritized items. Here we tested this by employing a working memory task with unequal item rewards to create uneven distribution of resources. Retro-cues then indicated which of three items could be dropped from memory. If reallocation occurs, then we should observe a larger benefit in memory when a high reward item versus a low reward item becomes irrelevant, since high reward items tie up more resources. We found that participants use both reward value and cue information to perform the task. There is better performance for items with higher reward values, F(2,90) = 10.85, p < .001, and for valid versus neutral cues, F(1,45) = 16.38, p < .001. However, our results show that removing a well-encoded item does not benefit performance more, t(45) = 0.05, p = .959 (JZS Bayes Factor = 6.245683). These findings challenge the dominant view that retro-cue benefits reflect reallocation of memory resources from removed items. Our results are more consistent with a benefit to cued items (such as retrieval protection or a general strengthening) that is independent of the memory strength of the de-prioritized item.
