AR spatial test for AD diagnosis

Rana Khoury

Sandy Rihana, USEK

Abstract

Alzheimer’s disease (AD) has been around for more than a hundred years, and so far, there is still no cure. However, the disease can be slowed down significantly if detected in its early stages. The problem is that it can only be detected through expensive means like MRI and PET scanning. Therefore, there is an increasing need to find new methods for the diagnosis of Alzheimer’s in its presymptomatic phase. This presents some challenges because the disease is thought to start way before clinical symptoms emerge. We therefore propose a new generation of realistic cognitive tests that can target early cognitive deficits of dementia much better than conventional tests used in the clinic. In particular, we propose to use Augmented Reality (AR) which can open the door for much more flexibility in designing and monitoring these tasks remotely.
