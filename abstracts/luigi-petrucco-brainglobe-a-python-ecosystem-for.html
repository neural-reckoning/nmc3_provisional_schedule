BrainGlobe: a Python ecosystem for computational (neuro)anatomy

Luigi Petrucco

Federico Claudi; Adam L. Tyson; Tiago Branco; Troy W. Margrie; Ruben Portugues

Abstract

Neuroscientists routinely perform experiments aimed at recording or manipulating neural activity, uncovering physiological processes underlying brain function or elucidating aspects of brain anatomy. Understanding how the brain generates behaviour ultimately depends on merging the results of these experiments into a unified picture of brain anatomy and function. We present BrainGlobe, a new initiative aimed at developing common Python tools for computational neuroanatomy, and its core package, the BrainGlobe atlas API (BG-AtlasAPI).
While digital high-resolution 3d atlases have been produced for several model organisms, the effective use of these atlases depends on the availability of an application programming interface (API) that enables researchers to develop software to access and query atlas data. Some atlases come with an API, but these are generally specific for individual atlases, and this hinders the development and adoption of open-source neuroanatomy software. BG-AtlasAPI overcomes this problem by providing a minimal Python interface to download and interact with atlases for the most common model organisms in systems neuroscience. BG-AtlasAPI is a small and extensible package that allows developers to develop software agnostic to the atlas and species, increasing adoption and interoperability of packages in neuroscience.
Finally, we showcase the usage of BG-AtlasAPI in other Python projects (for cell detection, atlas registration and visualisation) developed within the BrainGlobe ecosystem, to illustrate how it provides easy interoperability and support for brain atlases from different species.
